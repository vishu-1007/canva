Built a full-stack graphic design web application inspired by Canva, featuring a drag-and-drop canvas editor with AI image generation and real payment infrastructure.

1. Engineered an interactive canvas editor using Fabric.js with support for templates, text, shapes, and image layers
2. Integrated Replicate AI for text-to-image generation directly within the design workflow
3. Implemented Google OAuth, Github OAuth for authentication and Stripe payment gateway for premium feature access
4. Managed media uploads via Uploadthing and persisted user data with Neon (PostgreSQL)
5. Built and deployed a production-ready full-stack application handling end-to-end flows from design creation to export

Tech Stack: React, Node.js, Express, Neon (PostgreSQL), Fabric.js, Uploadthing, Replicate AI, Stripe, Google OAuth, Github OAuth
